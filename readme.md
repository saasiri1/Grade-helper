# Grade Helper

## Overview
This project is designed to assist users in managing and calculating grades effectively. It provides functionalities to input grades, calculate averages, and generate reports.

## Usage
1. **Setup**: Ensure you have the necessary environment to run the project. This may include a web browser or a specific runtime environment.
2. **File Structure**: The project consists of the following main files:
   - `content.js`: Contains the main logic for grade calculations and interactions.
   - `manifest.json`: Configuration file for the project.
3. **Running the Project**: Open the `content.js` file in a compatible environment and follow the instructions provided within the file to execute the functionalities.
4. **Features**:
   - Input grades manually.
   - Calculate average grades.
   - Generate reports based on the input data.
