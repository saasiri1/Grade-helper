# Grade Helper

## Overview
This project is designed to assist users in managing and calculating grades effectively. It provides functionalities to input grades, calculate averages, and generate reports.

## Usage
1. **Setup**: Ensure you have Google Chrome installed to run the project.
2. **File Structure**: The project consists of the following main files:
   - `content.js`: Contains the main logic for grade calculations and interactions.
   - `manifest.json`: Configuration file for the Chrome extension.
3. **Installing the Extension**:
   - Open Google Chrome and navigate to `chrome://extensions/`.
   - Enable "Developer mode" in the top right corner.
   - Click on "Load unpacked" and select the project folder.
4. **Running the Project**: Once loaded, the extension will be available in your Chrome browser.
